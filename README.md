## Description

This dataset compiles global statistics on digital privacy literacy and data protection behaviors (2020-2026). It includes four key variable domains: (1) Knowledge about data protection laws; (2) Use of privacy settings and protective behaviors; (3) Experiences of data breaches and personal data violations; and (4) Attitudes toward algorithmic surveillance.

Data compiled from VPNRanks, IBM Cost of a Data Breach Report, Eurobarometer, Pew Research, ENISA, and regional cybersecurity studies.

**Files included:** algorithmic_surveillance_attitudes.csv, data_breaches_annual_statistics.csv, privacy_knowledge_by_country.csv, privacy_settings_usage_behaviors.csv, top_countries_data_breaches.csv
